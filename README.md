# Visor 3D VR con Three.js, Bootstrap y SketchUp

Proyecto web para visualizar un modelo 3D exportado desde SketchUp en formato **GLB** utilizando **Three.js**.  
Incluye visualización interactiva, controles de cámara, soporte para modo VR WebXR y modo VR móvil tipo Cardboard.
Autor: Luis Enrique Cabrera Garcia 
ITP- Instituto Tecnologico de Pachuca
Materia : Desarrollo de Ambientes Virtuales 

---

## 📌 Descripción

Este proyecto permite cargar y visualizar un modelo 3D realizado en SketchUp dentro de una página web.  
El modelo se exporta en formato **`.glb`**, ya que es un formato recomendado para web porque puede contener geometría, materiales y texturas en un solo archivo.

La interfaz fue desarrollada con:

- HTML
- CSS
- JavaScript
- Bootstrap
- Three.js
- WebXR
- StereoEffect para VR móvil

---

## 🧩 Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura de la página |
| CSS3 | Diseño visual personalizado |
| JavaScript | Lógica del visor 3D |
| Bootstrap | Navbar, botones, tarjetas y diseño responsive |
| Three.js | Renderizado del modelo 3D |
| GLTFLoader | Carga del archivo `.glb` |
| OrbitControls | Movimiento interactivo de cámara |
| VRButton | Activación de VR con WebXR |
| StereoEffect | Modo VR móvil tipo Cardboard |
